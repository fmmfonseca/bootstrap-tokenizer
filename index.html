Tokenizer bootstrap-tokenizer.js

About

A basic bootstrap JavaScript plugin (with semicolons) to provide a tokenizer input field.

Download file

Example

Start typing in the field below and just hit space, enter or comma to create a new token.

You can also load default values.

It preserves the original input state.

Using bootstrap-tokenizer.js

Call the tokenizer via javascript:

$('.tokenizer').tokenizer()

Options

Name type default description
separator string ',' The character to use for separating the input values.
delimiters array [13,32,188] The key code(s) that trigger a new token.

Markup

Add data attributes to register an element with tokenizer functionality.

<input type="text" data-provide="tokenizer">

Methods

.tokenizer(options)

Initializes an input with tokenizer.
